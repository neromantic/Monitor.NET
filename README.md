# Monitor.NET

Netzwerk analysieren und Überwachen mit Monitor.NET
### Funktionalität
* IP-Addressen Bereich Scannen
* DNS-Name des Netzwerkgerätes holen
* Information über den Gerät bekommen
* Quieck-Connect-Aktionen
